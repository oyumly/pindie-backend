# Бэкенд для Pindie

Это серверная часть проекта «Pindie» — портала с рейтингами инди-игр. Проект написан в рамках курса «Веб-разработчик» от Яндекс Практикума. 

Сервер написан на Node.js (Express), в качестве базы данных используется MongoDB. Реализована система авторизации на JWT-токенах, а также полноценный API для работы с играми, категориями и пользователями.

## Стек технологий
- **Node.js** и **Express** — основа сервера и роутинг
- **MongoDB** и **Mongoose** — база данных и ODM
- **JWT** (`jsonwebtoken`) — авторизация пользователей (токены хранятся в HTTP-only куках)
- `bcryptjs` — хеширование паролей
- `cookie-parser` — парсинг кук
- `cors` — настроенные политики CORS для связи с фронтендом

## Основные фичи
- Регистрация и авторизация пользователей. Защита роутов (только авторизованные пользователи могут добавлять/изменять данные).
- CRUD-операции для игр (создание, чтение, обновление, удаление).
- CRUD-операции для категорий и пользователей.
- Связи в базе данных: у каждой игры есть список категорий и список проголосовавших пользователей.

## Запуск проекта локально

1. Склонировать репозиторий:
   ```bash
   git clone <ссылка_на_этот_репозиторий>
   ```
2. Установить зависимости:
   ```bash
   npm install
   ```
3. Для работы локально понадобится запущенная база MongoDB (по умолчанию сервер стучится на `mongodb://localhost:27017/pindie`).
4. Запустить сервер (используется nodemon для автоперезагрузки):
   ```bash
   npm start
   ```

По умолчанию сервер поднимется на порту `3000`.
